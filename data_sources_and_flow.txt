Definitions:
 - Broad product categories can be Tables, Bananas, Gaming PCs, Lip Gloss, Blenders, Shelves, TVs

Whenever a product's listing is scanned:
 - Scrape webpage for any product information
 - Query using Barcode IDs Anchor Table to get closest matching product details
    - Only use details that are missing from webpage's product information
 - Query any other DB APIs for additional relevant product information
 - Use all information about product

DB Anchor Table:
 - Match broad categories of products to barcode IDs
 - Whenever a product needs to be looked up:
    - Check which product in the Anchor Table is closest in concept
    - Use that closest product's corresponding DB domain to query for more detailed product information

Barcode IDs Anchor Table:
 - Match broad categories of products to barcode IDs
 - Whenever a product needs to be looked up:
    - Check which product in the Anchor Table is closest in concept
    - Use that closest product's corresponding barcode ID for more detail product information that is not available from original product source
